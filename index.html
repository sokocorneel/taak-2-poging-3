Hoe pagina aanpassen

html openen

om de pagina aan te passeen moet je eerst aan de code komen dit doe je door ctrl+shift+i. nu zou de html code rechts op je scherm moeten verschijnen.

html aanpassen

om te vinden wat je wilt aanpassen moet je met je muis over de zwarte pijltjes bewegen, als hetgene wat je wilt aanpassen is geselcteerd dan klik je op dat zwarte pijltje. dit blijf je herhalen tot er geen zwarte pijltjes merr open te kliken zijn. nu kun je normaal gezien ergens hetgene aanpassen wat je wilt aanpassen

soms selecteert het wat je wilt aanpassen maar kun je het niet aanpassen dan moet je gewoon verder zoeken naar hetgene wat je wilt aanpassen
